E-commerce SQL Analytics Project
Overview:
This project demonstrates end-to-end SQL skills by building and analyzing an e-commerce database. It covers database design, data manipulation, and advanced analytics using MySQL.
The project simulates a real-world e-commerce system with customers, orders, products, and payments, and generates meaningful business insights through SQL queries
Database Schema:

The database consists of the following tables:
Customers – Customer details and signup information
Products – Product catalog with categories and pricing
Orders – Customer orders and transaction details
Order_Items – Products included in each order
Payments – Payment details for each order

Basic SQL:

Filtering (WHERE, LIKE)
Sorting (ORDER BY)
Aggregations (SUM, AVG, COUNT)
Intermediate SQL:
Joins (INNER JOIN across multiple tables)
Grouping (GROUP BY)
Conditional logic (CASE statements)

Advanced SQL:

Views for reusable query logic
Stored Procedures for parameterized operations
Common Table Expressions (CTEs) for modular queries
Window Functions:
ROW_NUMBER()
RANK()
DENSE_RANK()
Running totals with SUM() OVER
LAG() and LEAD()
